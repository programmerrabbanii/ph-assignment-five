Important FAQ

What is the Document Object model (DOM)?

is a programming interface that represents HTML or XML documents as a tree structure. Each element, attribute, and piece of text in the document is a node, allowing JavaScript to interact with and manipulate the document

how do select an element from the DOM?

1.getElementById

2.getElementsByClassName

3.getElementsByTagName

4.querySelector

what is event delegation in the context of the dom and why is it useful

Event delegation is a technique that involves attaching a single event listener to a parent element instead of individual child elements. When an event occurs, it bubbles up to the parent, allowing you to check which child triggered it

how do you manipulate an elements attributes and styles the dom

1. setAttribute

2. getAttribute

3. removeAttribute
